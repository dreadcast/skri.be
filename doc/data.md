# Data

The data is your blog content.

Your blog content will be in a `data` folder. Each subfolder is a potential
blog entry (article).

## Structure
```
blog-root
|_ build
|_ data
|   |_ some-blog-post
|   |   |_ data.md
|   |   |_ image.jpg
|   |   |_ other-image.jpg
|   |
|   |_ some-other-blog-post
|       |_ data.md
|       |_ great-image.jpg
|
|_ package.json
|_ build.js
|_ dev.js
|_ purge.js
```
